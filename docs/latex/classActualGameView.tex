\doxysubsection{Detailed Description}
单机闯关游戏视图 

\char`\"{}\+Actual\+Game\+View\char`\"{} 这个名字来自早期开发，实际就是单机闯关模式的主视图。 联机模式的视图是 Online\+View 中的 Online\+Game\+View。
