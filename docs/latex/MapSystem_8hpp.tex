\doxysubsection{Detailed Description}
地图区域与摄像机系统（无限地图） 

基于区域网格实现无限地图：
\begin{DoxyItemize}
\item 世界被划分为 800x800 像素的区域（\+Map\+Region）
\item 玩家进入新区域时，自动检查周围3x3范围并生成敌人
\item 每个区域的敌人配置由其到原点的曼哈顿距离决定，越远越难
\item 已探索区域数影响全局难度缩放因子
\item 摄像机始终跟随玩家位置
\item 背景绘制半透明网格线标识区域边界 
\end{DoxyItemize}
