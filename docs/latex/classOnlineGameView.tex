\doxysubsection{Detailed Description}
联机游戏视图 

主机端和客户端共用此类，通过 is\+Host 区分行为：

主机端（is\+Host=true）：
\begin{DoxyItemize}
\item 运行完整游戏逻辑（敌人\+AI、投射物碰撞、掉落物拾取）
\item 接收客户端输入和攻击信号
\item 每帧广播完整游戏状态（玩家/敌人/投射物/掉落物）
\item 检查阶段性胜利，为所有玩家应用强化
\end{DoxyItemize}

客户端（is\+Host=false）：
\begin{DoxyItemize}
\item 发送输入状态（$\sim$30\+Hz）和攻击/翻滚信号
\item 接收并解析主机广播的游戏状态
\item 根据同步数据渲染游戏画面 
\end{DoxyItemize}
