\hypertarget{md_document_autotoc_md22}{}\doxysection{1. 项目概述}\label{md_document_autotoc_md22}
JRFirst\+Game 是一个基于 C++17 + SFML 3.\+x + SQLite3 构建的 RPG 闯关游戏。玩家在无限地图中击杀敌人、收集掉落物、提升属性，通过阶段性胜利不断强化角色。支持单机闯关和双人\+UDP联机合作两种模式。\hypertarget{md_document_autotoc_md23}{}\doxysubsection{1.\+1 核心玩法}\label{md_document_autotoc_md23}

\begin{DoxyItemize}
\item $\ast$$\ast$单机闯关$\ast$$\ast$：选择角色后进入无限地图，击杀敌人获得分数和经验，分数达到阈值触发阶段胜利，角色获得职业专属强化
\item $\ast$$\ast$双人闯关$\ast$$\ast$：一人创建房间（主机），另一人加入（客户端），共享同一地图合作对抗敌人，击杀分数共享
\item $\ast$$\ast$角色成长$\ast$$\ast$：六大属性随机生成，职业提供属性加成，升级增加全属性，掉落物可永久提升属性，分数提供倍率加成
\end{DoxyItemize}\hypertarget{md_document_autotoc_md24}{}\doxysubsection{1.\+2 技术选型}\label{md_document_autotoc_md24}

$\ast$$\ast$伤害计算$\ast$$\ast$： 
\begin{DoxyCode}{0}
\DoxyCodeLine{总伤害 = (武器基础伤害 + 属性加成 + 阶段强化伤害) × 分数倍率 × 暴击倍率}

\end{DoxyCode}



\begin{DoxyItemize}
\item 属性加成：剑→力量$\ast$0.8，法器→魔法$\ast$0.8，刺刀→敏捷$\ast$0.8
\item 分数倍率：1.0 + min(总分数/50 $\ast$ 0.\+1, 1.\+0)，范围\mbox{[}1.\+0, 2.\+0\mbox{]}
\item 暴击：基础5概率，刺客阶段强化后最高80，暴击倍率1.5起
\end{DoxyItemize}
