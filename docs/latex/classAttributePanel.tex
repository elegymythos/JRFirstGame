\doxysubsection{Detailed Description}
属性面板控件 

半透明黑色背景，显示角色完整属性信息：
\begin{DoxyItemize}
\item 六大属性（基础+加成=最终）
\item 职业、等级、经验、分数
\item 装备武器名称 Tab键切换显示/隐藏。 
\end{DoxyItemize}
