\doxysubsection{Detailed Description}
玩家角色 

继承 RPGEntity，增加以下系统：
\begin{DoxyItemize}
\item 属性与职业：基础属性、职业配置、等级数据
\item 武器系统：\+Weapon\+List\+Bridge 管理武器链表，可装备匹配职业的武器
\item 战斗属性：攻击伤害、范围、冷却，由武器和属性共同决定
\item 翻滚系统：\+Shift 触发，持续0.3秒，期间无敌，冷却0.8秒
\item 刀光动画：攻击时显示扇形剑气拖尾
\item 阶段胜利强化：根据职业不同，提升伤害/攻速/暴击/爆炸范围
\item 分数加成：每50分增加10全属性和速度，上限200\% 
\end{DoxyItemize}
