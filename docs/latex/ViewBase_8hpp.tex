\doxysubsection{Detailed Description}
视图基类，提供语言切换按钮等通用功能 

所有视图（菜单、登录、游戏、排行榜等）的公共基类。 提供：
\begin{DoxyItemize}
\item global\+Font 引用：所有子类共享的全局字体
\item 语言切换按钮：右上角显示当前语言，点击切换中/英文
\item language\+Changed 标记：语言切换后通知主循环重建当前视图
\item 三个纯虚函数：handle\+Event、update、draw，子类必须实现 
\end{DoxyItemize}
