\doxysubsection{Detailed Description}
工具函数：随机盐值生成和密码哈希 

提供两个安全相关的工具函数：
\begin{DoxyItemize}
\item \mbox{\hyperlink{Utils_8hpp_a3ad8e66739ba1f60a62abbc106cf855a}{generate\+Random\+Salt()}}：生成16字节随机数，输出为32字符的16进制字符串
\item \mbox{\hyperlink{Utils_8hpp_ac3a623f6d2431c7eba7d5c53c316ad1d}{hash\+Password()}}：使用纯\+C++实现的\+SHA-\/256，加盐后做10000轮迭代哈希
\end{DoxyItemize}

注意：此实现用于学习项目，生产环境应使用系统级加密库。
