\doxysubsection{Detailed Description}
联机游戏逻辑管理（简化版双人模式） 

维护多个 Net\+Player，处理输入、攻击和状态同步。 当前主要用于网络同步的基础框架，完整联机逻辑在 Online\+View 中。
