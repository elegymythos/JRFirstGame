\doxysubsection{Detailed Description}
文本输入框控件 

功能：
\begin{DoxyItemize}
\item 点击获取焦点（蓝色边框），点击其他区域失去焦点
\item 支持键盘输入（含中文\+UTF-\/8编码）
\item 密码模式：显示星号代替实际字符
\item 最大字符长度限制（按\+UTF-\/8字符数计算，非字节数）
\item 闪烁光标（焦点时每0.5秒切换）
\item 退格键删除（正确处理\+UTF-\/8多字节字符） 
\end{DoxyItemize}
