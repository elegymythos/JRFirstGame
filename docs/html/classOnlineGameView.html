Detailed Description

联机游戏视图

主机端和客户端共用此类，通过 isHost 区分行为：

主机端（isHost=true）：

  • 运行完整游戏逻辑（敌人AI、投射物碰撞、掉落物拾取）
  • 接收客户端输入和攻击信号
  • 每帧广播完整游戏状态（玩家/敌人/投射物/掉落物）
  • 检查阶段性胜利，为所有玩家应用强化

客户端（isHost=false）：

  • 发送输入状态（~30Hz）和攻击/翻滚信号
  • 接收并解析主机广播的游戏状态
  • 根据同步数据渲染游戏画面
