基于 C++17 + SFML 3.\+x + SQLite3 的塞尔达风格 RPG 闯关游戏，支持单机闯关、双人联机合作与 AI 自动游玩。\hypertarget{md_README_autotoc_md1}{}\doxysection{功能特性}\label{md_README_autotoc_md1}
\hypertarget{md_README_autotoc_md2}{}\doxysubsection{核心系统}\label{md_README_autotoc_md2}

\begin{DoxyItemize}
\item 用户系统：注册/登录，加盐哈希密码存储
\item 多角色：每用户最多 3 个角色槽位，支持创建/选择/删除
\item 五种职业：战士、法师、弓箭手、圣骑士、狂战士
\item 六大属性：力量/敏捷/魔法/智力/生命/幸运，正态分布随机生成
\item 等级系统：经验值升级，每级全属性 +2
\item 武器系统：C 语言单链表 + C++ 桥接层，三种武器（铁剑/炎之法器/暗影匕首）
\end{DoxyItemize}\hypertarget{md_README_autotoc_md3}{}\doxysubsection{游戏玩法}\label{md_README_autotoc_md3}

\begin{DoxyItemize}
\item 无限地图：基于区域网格，越远敌人越强
\item 五种敌人：史莱姆、骷髅刀斧手、骷髅弓箭手、巨人、\+Boss，各有不同 AI 和攻击方式
\item 战斗系统：扇形近战/远程投射物，刀光动画，暴击机制
\item 翻滚系统：\+Shift 键翻滚，翻滚期间无敌
\item 掉落物：击杀敌人概率掉落血瓶和属性提升道具（6 种）
\item 分数加成：每 50 分增加 10\% 全属性加成，上限 200\%
\item 阶段胜利：5 个阶段（100/1000/2000/5000/10000 分），每阶段强化角色并回满血
\item 暂停/存档：\+Esc 暂停，P 键中途存档
\end{DoxyItemize}\hypertarget{md_README_autotoc_md4}{}\doxysubsection{联机与社交}\label{md_README_autotoc_md4}

\begin{DoxyItemize}
\item 双人闯关：\+UDP 联机合作，主机-\/客户端架构
\item 排行榜：按等级和分数排名（\+Top 10）
\item 国际化：中/英双语切换（L 键）
