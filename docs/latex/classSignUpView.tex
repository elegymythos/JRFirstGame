\doxysubsection{Detailed Description}
注册视图 

用户输入用户名和密码，点击注册后：
\begin{DoxyEnumerate}
\item 检查用户名是否已存在
\item 生成 16 字节随机盐值
\item 用 SHA-\/256 + 盐值对密码做 10000 轮迭代哈希
\item 将用户名、盐值、哈希存入数据库 注册成功后自动跳转到登录页面。 
\end{DoxyEnumerate}
