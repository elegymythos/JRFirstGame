\doxysubsection{Detailed Description}
UI基础控件：\+Button、\+Input\+Field、\+Attribute\+Panel. 

提供三个可复用的 UI 控件：
\begin{DoxyItemize}
\item Button：矩形按钮，支持悬停变色和点击回调
\item Input\+Field：文本输入框，支持密码模式、\+UTF-\/8输入、最大长度限制
\item Attribute\+Panel：属性面板，\+Tab键切换显示，展示角色完整属性信息 
\end{DoxyItemize}
