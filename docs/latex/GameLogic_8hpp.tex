\doxysubsection{Detailed Description}
游戏核心逻辑：实体、属性、职业、等级、敌人、投射物、掉落物系统 

定义游戏所有核心数据结构和逻辑：
\begin{DoxyItemize}
\item 属性系统：六大属性（力量/敏捷/魔法/智力/生命/幸运）+ 职业加成 + 等级加成
\item 职业系统：战士/法师/刺客，各有属性加成和允许武器类型
\item 等级系统：经验值升级，每级属性加成
\item 敌人系统：四种敌人（史莱姆/骷髅刀斧手/骷髅弓箭手/巨人），各有差异化\+AI
\item 投射物系统：法师火球、弓箭手箭矢，沿直线飞行
\item 掉落物系统：血瓶和属性提升道具，击杀敌人概率掉落
\item 游戏实体：\+RPGEntity基类、\+Player玩家、\+Enemy敌人
\item 联机世界：\+Game\+World简化双人模式 
\end{DoxyItemize}
