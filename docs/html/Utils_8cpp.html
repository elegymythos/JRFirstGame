Detailed Description

工具函数实现：SHA-256哈希、随机盐值生成、密码哈希

包含纯C++实现的SHA-256算法（用于学习项目）， 以及generateRandomSalt()和hashPassword()两个公共接口。

Returns
64字符的16进制字符串（SHA-256哈希结果）

哈希过程（类PBKDF2）：

  1. 组合字符串 = salt + password + salt
  2. 第1轮：SHA-256(组合字符串)
  3. 第2-10000轮：SHA-256(上一轮哈希 + 组合字符串)
  4. 输出最终哈希的16进制表示
