\doxysubsection{Detailed Description}
敌人角色 

继承 RPGEntity，支持四种类型，各有差异化 AI：
\begin{DoxyItemize}
\item Slime：缓慢直线追踪
\item Skeleton\+Warrior：中速直线追踪，近战
\item Skeleton\+Archer：保持150-\/250距离，远程射击
\item Giant：慢速直线追踪，高血量近战 绘制为45度旋转的正方形（菱形）。 
\end{DoxyItemize}
