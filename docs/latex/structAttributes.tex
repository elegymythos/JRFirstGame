\doxysubsection{Detailed Description}
角色六大属性 

力量影响剑伤害，敏捷影响刺刀伤害和翻滚， 魔法影响法器伤害，智力预留，生命影响血量上限，幸运预留。 随机生成时使用正态分布（均值10，标准差4），钳制到\mbox{[}1,20\mbox{]}，总和上限80。
