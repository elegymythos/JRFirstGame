\doxysubsection{Detailed Description}
角色选择视图 

展示当前用户的3个角色槽位：
\begin{DoxyItemize}
\item 已占用槽位：显示职业、等级、分数，提供\char`\"{}选择\char`\"{}和\char`\"{}删除\char`\"{}按钮
\item 空槽位：显示\char`\"{}空槽位\char`\"{}，提供\char`\"{}新建角色\char`\"{}按钮 选择角色后，将槽位号存入 user\+\_\+settings 表，然后进入游戏。 
\end{DoxyItemize}
