Detailed Description

音频管理器 - 背景音乐(BGM)和音效(SFX)统一管理

单例模式，管理背景音乐和音效的加载、播放、音量控制。 BGM使用sf::Music流式播放，SFX使用sf::SoundBuffer+sf::Sound， 支持最多32个SFX通道轮换播放，避免音效被截断。 支持BGM淡入淡出效果。
