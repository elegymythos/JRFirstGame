\doxysubsection{Detailed Description}
SQLite数据库操作封装 

管理四张表：
\begin{DoxyItemize}
\item users：用户账户（用户名、盐值、密码哈希）
\item characters：角色数据（多角色，每用户最多3个槽位）
\item rankings：排行榜（按等级和分数排名）
\item user\+\_\+settings：用户设置（选中的角色槽位）
\end{DoxyItemize}

所有查询使用参数化语句防止\+SQL注入。 数据库文件为同目录下的game.\+db，程序启动时自动创建。
