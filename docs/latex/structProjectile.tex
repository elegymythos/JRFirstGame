\doxysubsection{Detailed Description}
投射物（法师火球、弓箭手箭矢） 

沿直线飞行，超过最大距离后消失。 法师火球命中后可产生爆炸范围杀伤（explosion\+Radius \texorpdfstring{$>$}{>} 0 时）。
