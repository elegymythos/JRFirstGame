Detailed Description

文本输入框控件

功能：

  • 点击获取焦点（蓝色边框），点击其他区域失去焦点
  • 支持键盘输入（含中文UTF-8编码）
  • 密码模式：显示星号代替实际字符
  • 最大字符长度限制（按UTF-8字符数计算，非字节数）
  • 闪烁光标（焦点时每0.5秒切换）
  • 退格键删除（正确处理UTF-8多字节字符）
