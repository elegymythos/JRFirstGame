Detailed Description

登录与注册视图

提供用户身份验证界面，包含登录（LoginView）和注册（SignUpView）两个视图。 登录时从数据库读取盐值和哈希，用 SHA-256 验证密码； 注册时生成随机盐值并存储哈希后的密码。 验证通过后，用户名会写入 main.cpp 中的 currentUsername 引用。
