\doxysubsection{Detailed Description}
联机大厅与联机游戏视图（双人合作闯关） 

设计思路：
\begin{DoxyItemize}
\item 主机端运行完整游戏逻辑（敌人、投射物、地图生成、分数计算、掉落物）
\item 客户端只发送输入，接收并渲染主机同步的游戏状态
\item 双人共享同一无限地图，各自独立移动和攻击
\item 主机通过扩展的 \mbox{\hyperlink{structGameState}{Game\+State}} 同步：玩家位置/血量、敌人、投射物、分数、掉落物
\item 联机模式与闯关模式玩法一致：掉落物、阶段胜利、完整\+UI
\end{DoxyItemize}

包含两个视图：
\begin{DoxyItemize}
\item Online\+Lobby\+View：联机大厅，输入\+IP和端口，创建或加入房间
\item Online\+Game\+View：联机游戏，主机运行逻辑+广播状态，客户端发送输入+渲染 
\end{DoxyItemize}
