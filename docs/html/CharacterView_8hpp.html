Detailed Description

角色创建与选择视图

提供两个视图：

  • CharacterCreateView：创建新角色，选择职业（战士/法师/刺客）， 随机生成六大属性，可重新随机，确认后存入数据库。
  • CharacterSelectView：展示已有角色列表（最多3个槽位）， 支持选择角色进入游戏、删除角色、在空槽位新建角色。
