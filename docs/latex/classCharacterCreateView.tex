\doxysubsection{Detailed Description}
角色创建视图 

流程：
\begin{DoxyEnumerate}
\item 自动随机生成六大属性（正态分布，范围\mbox{[}1,20\mbox{]}，总和上限80）
\item 用户可点击\char`\"{}重新随机\char`\"{}刷新属性
\item 选择职业：战士（力量+5/铁剑）、法师（魔法+5/火焰法器）、刺客（敏捷+5/暗影刺刀）
\item 确认创建，自动分配空槽位，存入数据库 
\end{DoxyEnumerate}
