\doxysubsection{Detailed Description}
模式选择视图 

登录成功后首次进入时显示2秒欢迎动画，之后显示按钮：
\begin{DoxyItemize}
\item 创建角色 / 选择角色（取决于是否已有角色）
\item 闯关模式（进入角色选择或创建）
\item 双人闯关（进入联机大厅）
\item 排行榜
\item 返回 
\end{DoxyItemize}
