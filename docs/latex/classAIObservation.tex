\doxysubsection{Detailed Description}
AI 观测编码器 

将 \mbox{\hyperlink{structAIGameState}{AIGame\+State}} 编码为固定维度的浮点向量供 ONNX 模型推理， 并将模型输出的动作索引解码为 AIAction。

编码方式与 Python 训练环境 rl/game\+\_\+env.\+py 完全一致。
